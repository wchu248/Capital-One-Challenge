Yelp's Fusion API

Yelp's Fusion API was used to help users pick a restaurant. The input to the questionnaire will form a query that will search all the restaurants near your location and recommend the perfect restaurants for you!

face

User Friendly

By using the framework of Materialize Design, an interface was created that is clear to follow and allows our users to find restaurants with ease.

location_on

Location Based

Your location is obtained via HTML5 Geolocation and used as your default location to search for restaurants near you. All you have to do is set a certain radius (in miles) of how far you're willing to travel for food and we'll find you a restaurant!
